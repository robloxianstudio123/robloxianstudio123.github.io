How to Edit the Pagefile Size

If you wish to change the size of the pagefile (`pagefile.sys`) manually, follow these steps:

  1. Press Windows Key + Pause/Break to open the System Properties window.
  2. Click on the Advanced system settings link on the left-hand side.
  3. In the System Properties window, go to the Advanced tab.
  4. Under the Performance section, click the Settings button.
  5. In the Performance Options window, go to the Advanced tab.
  6. Under the Virtual Memory section, click the Change button.
  7. Uncheck the box that says Automatically manage paging file size for all drives.
  8. Now, you can set the custom size for the pagefile by selecting a drive, and then selecting either Custom size or System managed size.
  9. Click Set to apply your changes, and then click OK.
  10. Restart your computer for the changes to take effect.
